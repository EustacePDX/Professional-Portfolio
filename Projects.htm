Research Assistant on Bioinformatics program

- September 2014 - Decmber 2014

Done as an independent study under my mentor professor (CS 498). This project involved verifying a previously completed study of GPU calculations on fractal dimensions of biofilm images.

Our goal was to implement a software called BIP (biofilm image processing) on the GPU using C++ and CUDA code. The original code for BIP is now only binary and therefore had to be reverse engineered. In its current state this software runs a box counting algorithm to calculate the fractal dimensions of images of biofilms. It was my job to verify this code and place it online in a user freindly format.

* Please Note: This is a project which is still in development at Eckerd College, therefore sample code is not avalible.

Java Geometry Package

- February 2014 - May 2014

Developed with a partner this java package was built as an extension to the Java API libraries. Designed to allow programmers ease of access to Euclidean geometric functions. Simply by importing our package, using our dot extensions, and providing the prompted input, programmers can run calculations on many geometric shapes.

The package provides functions for the creation and manipulation of both two dimentional and three dimensional shapes. Using inheritance from a shape class programmers define the shape by its dimension, number of sides, and vertices in the Eucliden plain. Using those definitions programmers are also given access to dot extensions that allow for the rotation, translation and reflection of all shapes.

iPhone Application in Objective C for iOS

- September 2013 - December 2013

Designed and engineered with a partner, this iPhone application was created to show mastery of GUI design concepts.

Built through Xcode, the application uses NStimers to detect when the user is holding down one of four buttons. Each button starts the animation of a character moving through out a room in one of the four cardinal directions by altering sprites on an image view element and shifting a background image view. When the background detects out of bounds coordinates movement is prohibited.

Animation

- February 2012 - May 2012

This short animation was built in a program called POVRAY, it was designed with a partner to demonstrate proficiency with complex graphics concepts.

This animation lasts sixteen seconds and takes advantage of the difference and union functions building complex final objects from shape primatives. These final objects are then image and bump map texturized. The clock function is mainpulated to translate the camera and final objects to create an animation. The final product is rendered at a six hundred by eight hundred resolution running at twenty four FPS.

Java Swing Game

- September 2012 - December 2012

A multiclass user input based game designed to demonstrate mastery of data structures. Created in a group of three this game makes use of the Java Swing libraries for its GUI.

Program creates a craft which uses the input controls. When the craft detects collision, movement is prevented. Some coordinates open a battle window. These windows contatin buttons allowing players to add and subtract from two different integers. If throughout the game the integer representing health never reaches zero a special congratulatory window is opened, after which the system exits.
